使用教程

  1. Android(Eclipse和ADT Bundle)开发环境
    打开SDK Manager的设置页, mac 没有入口，需要通过 cmd + , 打开。

  1. 给Android Stuido / Intellij IDEA 配置代理
    打开Preference页面，找到HTTP Proxy, 选择Manual proxy configuration
    Host Name: hx.gy, Port Number: 1080

  2. 可以自己在浏览器中设置红杏代码,但是红杏公益代理只支持部分域名代理
